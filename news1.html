AMD Combat Crate: Un pack de CPU + Placa Base y GPU para limpiar stock

14/04/2018 13:38

AMD anunciará próximamente lo que denomina el Combat Crate (Caja de Combate), un nombre muy épico para un mero pack de hardware que incluye componentes de anterior generación para ir limpiando el stock.


En concreto, el AMD Combat Crate se conformará por una CPU AMD Ryzen 7 1700 o AMD Ryzen 5 1600 junto a una placa base MSI B350 Tomahawk basada en el chipset AMD B350, junto a una MSI Radeon RX 580 ARMOR que ayudará a romper el equilibrio precio/rendimiento de este ‘bundle’, a la par de intentar ayudar a MSI de limpiar el stock de placas base B350 antes de que lleguen al mercado las B450. Como era de esperar, en lo que respecta a disponibilidad o precio ni una palabra, y quizás este bundle no sea una mala idea, pero añadir una gráfica Radeon RX 580 a la ecuación, no es la mejor de las ideas conociendo que los precios de estos modelos son lo suficientemente desorbitados para no compensar su compra.
